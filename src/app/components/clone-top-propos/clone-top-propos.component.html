Vinted Clone

Clone de la plateforme qui est un marché en ligne communautaire qui permet de vendre, d'acheter, et d'échanger des vêtements et accessoires d'occasion. La société est créée en 2008 à Vilnius, capitale de la Lituanie par un couple originaire du pays.

Les technologies utilisées :

Front :

  • Angular
  • Bootstrap

Back :

  • NodeJS
  • Typescript
  • GraphQL

Database :

  • MongoDB
